# VariaType — Lab README

## Overview

The **VariaType** lab focuses on web application security and multi-stage exploitation through file processing vulnerabilities. The goal is to move from initial access to full system compromise by chaining weaknesses across different components.

---

## Objectives

- Identify attack surface
- Gain initial access
- Perform privilege escalation
- Achieve full system compromise

---

## Focus Areas

- **Web Security**: file handling and input validation  
- **File Processing**: unsafe parsing and trust boundaries  
- **Vulnerability Chaining**: multi-step exploitation  
- **System Analysis**: understanding backend and automation  
- **Privilege Escalation**: sudo misconfigurations and script abuse  

---

## Skills Developed

- Enumeration (services, directories, vhosts)  
- Exploitation strategy and pivoting  
- Shell access and handling  
- Linux fundamentals (permissions, filesystem)  
- Privilege escalation techniques  

---

## Key Takeaways

- File uploads are a critical attack surface  
- Exploits often require chaining vulnerabilities  
- Automated processes (e.g., cron jobs) can be abused  
- Misconfigurations frequently lead to full compromise  
- Understanding system behavior is essential
