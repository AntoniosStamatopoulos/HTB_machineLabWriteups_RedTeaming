# Useful Files

This directory contains scripts, payloads, and helper files used during the exploitation process of the target machine.

## Contents

- **Exploit Scripts**
  - Custom or public exploits used to gain initial access

- **Payloads**
  - Reverse shells, bind shells, and command execution payloads

- **Hash Files**
  - Extracted hashes for cracking (e.g., `hash.txt`, `hash_pbkdf2.txt`)

- **Wordlists / Resources**
  - Dictionaries or supporting files used for password cracking

## Purpose

These files were used to:
- Automate exploitation steps
- Store intermediate data (e.g., hashes)
- Facilitate privilege escalation
- Improve efficiency during the attack process

## Notes

- Replace placeholders such as `<YOUR_IP>` when reusing scripts
- Some files may require specific dependencies (e.g., Python modules, hashcat)
